Hide Sticky Header when you scroll down and appear when you scroll up again.

== Description ==

This plugin make your header sticky on top of page. Also add feature hide header when scroll down. Hide the amount of headers by pixels you scroll.
This plugin only works with astra themes.
== Installation ==

1. Upload the "hide-when-scroll-down" directory to your "wp-content/plugins" directory.
2. In your WordPress admin, go to PLUGINS and activate "Sticky Header & Hide When Scroll Down"
3. Party!

== Frequently Asked Questions ==

= How do I use this plugin? =

Just download and activate it.

= How to uninstall the plugin? =
 
Simply deactivate and delete the plugin.
